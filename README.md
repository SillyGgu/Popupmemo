# 팝업 메모장

## 소개

본 확장은 채팅 환경에서 **팝업 메모장** 기능을 제공하여, 쉽게 찾아야하는 정보를 기록하거나 대화에 필요한 상주 요소를 유연하게 관리할 수 있도록 설계되었습니다. 특히, 캐릭터와 페르소나 각각에 대한 **말풍선** 시스템을 통해 대화의 흐름과 관계없는 커스텀 요소를 제공합니다.

본 확장팩의 기능 및 코드를 참고하여 게시글 또는 제작에 활용하실 경우, 다음 정보를 반드시 명시해 주십시오: 확장명(팝업 메모장)과 출처 링크(해당 확장팩을 공유하거나 설명한 원제작자 게시글의 링크를 명시해 주시면 됩니다. Github링크 X)

-----

## ✨ 주요 기능 및 장점

### 1. 자유로운 위치 및 크기 설정

  * **자유로운 이동 및 저장:** 메모장 팝업 창을 마우스로 드래그하여 화면의 어느 곳이든 원하는 위치에 배치할 수 있으며, 설정된 위치와 크기는 자동으로 저장됩니다. (모바일 한정 : 채팅창 사이즈로 자동으로 맞춰집니다)
  * **자유로운 크기 조절:** 팝업 창의 크기를 사용자가 원하는 대로 늘리거나 줄일 수 있습니다.
  * **항상 위 표시:** 메모장이 다른 화면 요소에 가려지지 않도록 `z-index`가 높게 설정되어 항상 화면 위에 표시됩니다. 설정창보단 아래에 있어 설정창을 가리지 않습니다.

### 2. 커스텀 배경 및 시각 설정

  * **배경 투명도:** 메모장 배경의 투명도를 0.0부터 1.0까지 세밀하게 조절하여 메모장 내 글씨를 읽는데 방해하는걸 줄입니다.
  * **커스텀 배경 이미지:** 원하는 이미지 URL을 등록하여 메모장 팝업의 배경으로 사용할 수 있습니다.

### 3. 말풍선 꾸미기

  * **다중 말풍선 등록:** 캐릭터와 페르소나 각각 최대 3개까지의 말풍선 내용을 등록할 수 있으며, 설정된 메모들은 순차적으로 회전하며 표시됩니다.
  * **캐릭터별 설정 관리:** 글로벌 설정 외, 말풍선 내용은 **캐릭터별로 개별적으로 저장**이 가능합니다.
  * **색상 커스터마이징:** 캐릭터 및 페르소나 말풍선의 배경 색상을 원하는 대로 설정할 수 있습니다.
  * **아바타 이미지 오버라이드:** 메모장에 표시되는 캐릭터 및 사용자 아바타 이미지를 별도의 URL로 변경할 수 있습니다.

### 4. 사용 편의 기능

  * **팝업 클릭 무시**
  * **간편 활성화/토글:** 확장 기능 메뉴를 통해 메모장 활성화/비활성화 및 메뉴 버튼 표시 여부를 쉽게 제어할 수 있습니다.
  * **모바일 최적화:** 모바일 환경(768px 이하)에 맞춰 자동으로 크기와 위치가 조정되어 사용 편의성을 높였습니다.
  * **메모 데이터 이동, 복사:** 확장 설정에서 메모 데이터를 한꺼번에 관리할 수 있습니다.
